# coletasp

To install dependencies:

```bash
bun install
```
in one terminal:
```bash
cd frontend
bun install
bun dev
```
in another terminal:
```bash
cd backend
bun install
bun dev
```

This project was created using `bun init` in bun v1.2.5. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
